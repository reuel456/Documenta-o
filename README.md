## 🤣Comandos mais utilizados 
pwd: Mostra o caminho da pasta atual.
ls: Lista arquivos e pastas.
cd: Entra em pastas.
mkdir: Cria pastas.
touch: Cria arquivos.
cp: Copia arquivos ou pastas.
mv: Move ou renomeia arquivos ou pastas.
rm: Remove arquivos (cuidado!).
nano / vim: Editores de texto internos.

## 👌Qual foi a minha dificuldade 

Ao utilizar os editores de texto no terminal, foram observadas as seguintes dificuldades:

- **Nano**: 
  - Dificuldade inicial para memorizar os atalhos de salvamento (`Ctrl+O`) e saída (`Ctrl+X`).
  - Pouca visibilidade de recursos avançados de edição.

- **Vim**:
  - Curva de aprendizado acentuada, especialmente na alternância entre os modos *Insert*, *Normal* e *Visual*.
  - Dificuldade para sair do editor sem salvar (`:q!`) ou salvar e sair (`:wq`).
  - Necessidade constante de consultar a documentação (`:help`) para comandos básicos.

Esses desafios reforçaram a importância da prática contínua para ganhar fluência com ferramentas de linha de comando.
